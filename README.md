# Password-Strength-Analyzer



## Overview
A Python GUI application that analyzes password strength 
and helps users create secure passwords.

## Features
- Real-time password strength scoring
- Common password dictionary check
- Secure random password generator
- Password visibility toggle
- Strength feedback with color indicators
- Password tips and guidelines

## Tools & Technologies
- Python
- Tkinter (GUI)
- String & Random libraries

## How to Run
1. Clone this repository
2. Run: python final_project.py
